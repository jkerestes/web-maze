# Web Maze

A web-based maze game built with HTML, CSS, and JavaScript.

## Project Structure

```
web-maze/
├── index.html      # Main HTML file
├── css/
│   └── style.css   # Styling
├── js/
│   └── app.js      # JavaScript logic
└── README.md       # This file
```

## Getting Started

1. Open `index.html` in your web browser
2. Start building your maze!
